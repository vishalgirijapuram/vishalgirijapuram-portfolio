F1 Racing Database Analytics

The project focused on migrating and transforming historical racing data into a scalable cloud environment, enabling interactive dashboards and detailed performance analysis for data-driven decision-making.

  • Technologies Used: Python, Azure Data Factory, Snowflake, Power BI
  • Role:The project migrated historical F1 racing data to Snowflake using Azure Data Factory, enabling secure and scalable cloud integration. It processed large datasets, designed optimized data models, and created interactive Power BI dashboards for dynamic analysis of race trends and performances.

Predictive Maintenance for IoT Devices

The Predictive Maintenance for IoT Devices project enabled proactive maintenance of IoT devices, reducing downtime and repair costs through data-driven insights and timely interventions.

  • Technologies Used: ADF, Databricks, Datalake, Power BI, Python
  • Role:The solution utilized Azure Data Factory for ETL, Azure Data Lake for scalable storage, and Azure Databricks for model training to predict maintenance needs. Predictive models in Python were deployed with real-time visualizations in Power BI, supported by automated CI/CD pipelines for continuous improvements.

Sales Performance Dashboard

The Sales Performance Dashboard project provided actionable insights into sales trends and performance, enabling stakeholders to make data-driven decisions to optimize sales strategies.

  • Technologies Used:ADF, Azure SQL Database, Azure Synapse, Power BI
  • Role:The project used Azure Data Factory to integrate and process sales data from multiple sources, storing it in Azure SQL Database for transformation and aggregation. Advanced queries and a data model in Azure Synapse powered detailed analysis, with Power BI dashboards featuring interactive filters and drill-down features for dynamic sales performance visualization.

Home Automation using IoT

The Home Automation using IoT project enabled remote monitoring and control of home devices, enhancing convenience, energy efficiency, and user experience through intelligent automation.

  • Technologies Used:Arduino IDE, Cloud service (Adafruit IO and ThingSpeak)
  • Role:The project utilized Adafruit IO for real-time device monitoring and control, and ThingSpeak for cloud-based data logging and analytics. IoT devices were integrated to automate actions, and a user-friendly interface was developed for seamless control via web and mobile applications.
